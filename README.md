# 🏨 Hotel Booking Web App (MERN Stack)

A full-stack **hotel booking web application** built with **MongoDB, Express, React, and Node.js (MERN)**. Users can browse rooms, view amenities, check pricing, and make reservations online. Admins can manage rooms, bookings, and users through a dashboard.  

## 🚀 Features
- Responsive and modern UI with React
- Room listings with images, amenities, and pricing
- User authentication (sign up / login)
- Booking system with MongoDB storage
- Admin dashboard to manage rooms, users, and bookings
- RESTful API built with Express & Node.js
- Mobile-friendly and accessible design  

## 🛠️ Technologies Used
- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT / bcrypt
